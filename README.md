# bbc64
Port of BBC Micro OS (MOS) and BBC BASIC to Commodore 64. Currently modifying the MOS ROM
source code to work with the C64 hardware.

Build with the [ACME Assembler](https://sourceforge.net/p/acme-crossass/code-0/HEAD/tree/).
You'll need a version of ACME built from source code to get the feature that enables calling
a macro passed as an argument to another macro, which is required to build the BASIC tables.

Original [BBC BASIC annotated source code](https://github.com/ivop/bbc-basic).

Original [MOS 1.20 annotated source code](https://tobylobster.github.io/mos/index.html).
